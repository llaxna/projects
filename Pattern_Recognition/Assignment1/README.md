# Image Processing
In this assignment, you will work with images using OpenCV library. 

1.    Read the road.jpg image using OpenCV. Print it to the screen.
2.    Print the shape of your image. What are the height, width, and number of the channels?
3.    Read the image as a grayscale, then print the shape. Is there a difference in the result? 
4.    Slice/crop the white biggest truck in the image and show the cropped image, print the pixel intensities values of the cropped ROI, then change its pixel values to 0, attach the new value to the old image, and print the modified one.
5.    Resize the image by 0.5, then rotate it by 45 degrees clockwise, and show the result.
