# Text Analysis and Visualization using NLTK and WordCloud
This script processes the *Alice in Wonderland* text dataset, tokenizes it into sentences, analyzes sentence lengths, and generates a word cloud visualization.

## Requirements
Ensure you have the following Python libraries installed:
- nltk (Natural Language Toolkit)
- pandas
- matplotlib
- wordcloud

## Dataset: Alice in Wonderland
This script works with the *Alice in Wonderland* dataset. Ensure that test.txt contains the full text of *Alice's Adventures in Wonderland* by Lewis Carroll.

## Output
- A histogram visualizing sentence length distribution.
- A word cloud representation of the most frequent words in Alice in Wonderland.
