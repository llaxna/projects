# Projects and Assignments - PSUT
This repository contains a collection of projects and assignments completed during my academic journey at **Princess Sumaya University for Technology (PSUT)**. It includes work from courses in various areas, including Object-Oriented Programming (OOP), Data Structures, Algorithms, Pattern Recognition, Database Systems, Machine Learning, Robotics, and more.

## 📂 Repository Overview
The repository is divided into folders corresponding to specific courses and their associated projects/assignments. Below is an overview of the main courses and projects:

### Course Overview
- **Data Structures and Algorithms (11212)**: Complexity analysis, data structures (lists, stacks, queues, trees, heaps), and algorithms (searching, sorting).

- **Database Systems (11323)**: Database design, DBMS components, data modeling, relational databases, normalization, query languages, and integrity constraints.

- **Object-Oriented Programming (11206)**: Concepts like data abstraction, encapsulation, inheritance, polymorphism, and problem-solving using OOP principles.

- **Computer Architecture and Machine Learning (14350)**: Introduction to machine learning, supervised/unsupervised learning, deep learning, AI applications (computer vision, data mining, bioinformatics).

- **Pattern Recognition (14452)**: Pattern recognition systems, preprocessing, feature extraction, supervised/unsupervised learning, object classification, and recognition methods.

- **Robotics Programming (14456)**: Modeling, designing, planning, and controlling robot systems using geometry, kinematics, and control algorithms.

- **Algorithm Design and Analysis (11313)**: Techniques for designing and analyzing algorithms, including divide-and-conquer, greedy algorithms, dynamic programming, and NP-Completeness.

## 🛠 Technologies Used
- **Programming Languages**: Python, C++
- **Libraries & Frameworks**: OpenCV, scikit-learn, NumPy, Pandas, Matplotlib
- **Tools**: Jupyter Notebook, Google Colab, Visual Studio Code
